newブランチ
　bot.c
　　zombie2
　　　detectZombie2を実行して値を配列に入れる。
　　　z2printを実行する。
　　　ゾンビを認識したら0、認識しなかったら1を返す。
　　z2print
　　　detectZombie2の値からゾンビの場所を表示する。
　　z2judge
　　　zombie2を実行し、値によって以下の動作を行う。
　　　　中央近・中央中ならそのまま攻撃
　　　　左下近・左上近なら後ろに移動し左を向いて攻撃
　　　　右下近・右上近なら後ろに移動し右を向いて攻撃
　　　　左下中・左上中なら左を向いて攻撃
　　　　右下中・右上中なら右を向いて攻撃
　　　　左下遠なら左を向いて攻撃、右下遠なら右を向いて攻撃
　　　　中央遠ならそのまま攻撃
　　　　左上遠なら左を向いて前にダッシュして攻撃
　　　　右上遠なら右を向いて前にダッシュして攻撃
　　notz2
　　　ゾンビを認識していない間、ダッシュと右を向く動作を繰り返す。
　　reset
　　　視点をリセットする。
　　attack1
　　　attackLeft_Longで攻撃を10回行う。
　　main
　　　最初にresetを行い、その後はz2judgeとnotz2を繰り返す。

　python/minecraft
　　pushKey.py　→　void pushKey(char* key);
　　　特定のキーを0.05秒押す。
　　　視点操作などに使用する。
　　clickLeft_Long.py　→　void attackLeft_Long(void);
　　　攻撃を10回行いながら、少し後ろに下がる。
　　moveDash1.py　→　void dash1(void);
　　　ダッシュを0.8秒行う。
　　moveDash2.py　→　void dash2(void);
　　　ダッシュを2秒行う。

　control.c/control.h
　　moveDash1、moveDash2用に変更。
